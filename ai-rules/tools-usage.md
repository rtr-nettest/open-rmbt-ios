# Tools Usage Guide

Leverage the `xcodebuild` command for building and testing code.

## MCP tools

Currently, we don't use any MCP tools.

## CLI tools

### `axe`: Verify implemented behavior using iOS simulator

When working on an iOS app, you can use the `axe` CLI tool to run the app inside the iOS simulator, tap on UI elements, and verify the feature you've built works properly.

If you're not sure, use `axe --help` to see usage and available options.
